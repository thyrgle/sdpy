# SDPy Slides

This repository contains slides by Christopher Sumnicht for the SDPy in-person meetings.

# View the Slides

To view the slides make sure [mkslides](https://github.com/MartenBE/mkslides) is  installed. This can be done with [pipx](https://pipx.pypa.io/stable/how-to/install-pipx/):

```
pipx install mkslides
```

Once installed you can run:

```
cd sdpy
mkslides build sdpy_slides.md
```

This will generate a `site/index.html` which can be opened up in the browser.
